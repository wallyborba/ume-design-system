# Ume Design System

Design System oficial da Ume — foundations, components e patterns para produtos digitais.

**Site:** [ume-design-system.vercel.app](https://ume-design-system.vercel.app)

## Estrutura

| Arquivo | Conteúdo |
|---------|----------|
| `index.html` | Overview — visão geral do DS |
| `foundations.html` | Cores, tipografia, espaçamento, grid |
| `components.html` | Componentes UI (shadcn/ui) |
| `patterns.html` | Padrões de composição |
| `globals.css` | CSS global de referência para devs |
| `tokens.json` | Design tokens (JSON) |
| `tailwind.config.ts` | Configuração Tailwind de referência |
| `fonts/` | Aeonik e Aeonik Fono (OTF) |
| `logos/` | Logos Ume (SVG) |
